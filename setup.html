3. Checklist de despliegue

1
Crear Google Sheets con 3 hojas: Usuarios, Cotizaciones, Tokens. Importar sagatreequote_google_sheets_template.xlsx para tener la estructura lista.
2
En script.google.com, crear nuevo proyecto, pegar Code.gs y reemplazar SPREADSHEET_ID.
3
Implementar → Nueva implementación → Aplicación web. Ejecutar como: Yo. Acceso: Cualquiera.
4
Pegar la URL generada en assets/js/config.js → campo APPS_SCRIPT_URL.
5
Usar el generador arriba para obtener el hash de la contraseña admin. Insertarlo en la hoja Usuarios.
6
git push origin main → GitHub Pages despliega automáticamente vía .github/workflows/deploy.yml.
